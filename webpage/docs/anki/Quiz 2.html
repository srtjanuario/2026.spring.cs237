Bayes Rule
Given events \(A,B\) with \(\mathbb{P}[A] > 0, \mathbb{P}[B] > 0\), Bayes rule allows you to switch the order of conditional probability.\[\mathbb{P}[A \mid B] = \frac{\mathbb{P}[B \mid A]\mathbb{P}[A]}{\mathbb{P}[B]}\]This is easy to derive from the definition of conditional probability and the chain rule.
Bonus card: Bayesian update 
Life is uncertain. To model this, we all have probability distributions in our minds. Maybe we think, for example, there is a 95% chance that if we leave home at 9am we will arrive in class on time: this is \(\mathbb{P}[A] = 0.95\), where \(A\) is the event that I'm on time. This initial distribution is called our prior. 

These distributions are not very useful if they are static and don't update over time. If I leave at 9am and then see that the T is not going to come for 20 minutes, I need to update my distribution based on new information and perhaps change my plan; i.e. I need to compute \(\mathbb{P}[A \mid D=20]\), where \(D\) is the random variable that displays the estimated arrival time of the T. 

Based on common sense and past experience, maybe I can estimate \(\mathbb{P}[A \mid D = 20] < 0.05\), and find another way to get to class.

This simple process of updating your distribution based on events we observe is called a Bayesian update (or sometimes, "updating your priors"). Despite the fancy name, it is a basic thing we do over and over in our daily lives. 

It is also something people often fail to do when they should! If you're rooting for a hockey team and you're absolutely sure they'll win (maybe you just bet your friend $20 they will), but halfway through the game they are down 5 points, you should probably update your priors instead of doubling down.

Bonus card: do continuous random variables exist in the real world?
This is a deep question, and is no different than asking whether the universe is continuous or discrete.

Let's suppose for sake of argument that time and space are continuous. Then, continuous random variables really do exist: the time it takes for you to walk 10 steps is a continuous quantity which could take the value of any real number. 

However, as humans, we can never observe the value a continuous random variable takes on. We can only observe discretizations of their values. I can never know exactly how long it took you to walk 10 steps, because even atomic clocks cannot accurately measure time in increments smaller than \(10^{-18}\) seconds.

Nevertheless, continuous random variables are extremely useful, because they allow us to compute the probabilities of events that occur in the real world - which, for all intents and purposes, acts like a continuous one.

CDF
Cumulative distribution function. Given a random variable \(X\) which may be continuous or discrete, the CDF is the function \(F(x) = \mathbb{P}[X \le x]\). 
Chain rule (probability)
Given two events \(A,B\) with \(\mathbb{P}[A] > 0\) and \(\mathbb{P}[B] > 0\):
\[\mathbb{P}[A \cap B] = \mathbb{P}[A \mid B]\mathbb{P}[B] = \mathbb{P}[B \mid A]\mathbb{P}[A]\]
Definition of conditional probability
Computes the probability of an event \(A\) given, or conditioned on, an event \(B\) with \(\mathbb{P}[B] > 0\). It is defined as:
\[\mathbb{P}[A \mid B] = \frac{\mathbb{P}[A \cap B]}{\mathbb{P}[B]}\]
Discrete versus continuous random variables
A discrete random variable is one whose range contains a countable number of values. For example, it could count the number of times you need to flip a coin to obtain heads. In this example, the range is infinite but countable. It could also have finite range, as all finite sets are countable: for example, the number of heads obtained after flipping 10 coins.

A continuous random variable is one whose range contains an uncountable number of values. For example, if we pick a random point on the real line between 0 and 10, the value of the point we pick is a continuous random variable. This is because the set of real numbers is uncountable. 
How to obtain a CDF from a PDF
Integrate it. \(F(x) = \int_{-\infty}^x f(x) dx\).
How to obtain a CDF from a PMF
Sum. \(F(x) = \sum_{-\infty}^x f(x)\).
How to obtain a PDF from a CDF
Differentiate the CDF. \(f(x) = F'(x)\).
How to obtain a PMF from a CDF
For a random variable \(X\), \(f(x) = F(x) - F(y)\), where \(y\) is the largest number in the range of \(X\) smaller than \(x\). 
Law of total probability
Given an event \(A\) and an event \(B\) with \(\mathbb{P}[B] > 0\), the law of total probability states:
\[\mathbb{P}[A] = \mathbb{P}[A \mid B]\mathbb{P}[B] + \mathbb{P}[A \mid \overline{B}]\mathbb{P}[\overline{B}]\]
More generally, given an event \(A\) and an partition of the sample space \(B_1,\dots,B_k\), the law of total probability states:
\[\mathbb{P}[A] = \sum_{i=1}^k \mathbb{P}[A \mid B_i]\mathbb{P}[B_i]\]
Note this also works with a countably infinite partition. 
PDF
Probability density function, usually denoted \(f(x)\) or \(f_X(x)\) for a random variable \(X\). This is for a continuous random variable and measures the density of the random variable at \(x\).

Notice it does not indicate the probability that \(X=x\), as this probability is 0 since \(X\) is continuous. The only way to get statements about probabilities from PDFs is to take integrals.
PMF
Probability mass function, usually denoted \(f(x)\) or \(f_X(x)\) for a random variable \(X\). This is for discrete random variables and, for a random variable \(X\), outputs the probability that \(X=x\).
Probability of a single outcome in a continuous probability space
Every individual outcome in a continuous probability space occurs with probability 0
Properties of PDF
A function \(f: \mathbb{R} \to \mathbb{R}\) is a PDF if
  • \(f(x) \ge 0\) for all \(x \in \mathbb{R}\)
  • \(\int_{-\infty}^\infty f(x) dx = 1\)

A small technicality that we won't test you on is we also technically need \(f\) to be Lebesgue measurable so that the integral over \(\mathbb{R}\) (and other sets) is well defined.
Properties of PMF
A function \(f: \mathbb{R} \to \mathbb{R}\) is a PMF if:
  •  \(f(x) \ge 0\) for all \(x \in \mathbb{R}\)
  • \(\sum_{x \in \mathbb{R}} f(x) = 1\)
Properties of a CDF
A function \(F: \mathbb{R} \to \mathbb{R}\) is a CDF if
  • \(F(x)\) is non-decreasing
  • \(\lim_{x \to -\infty} F(x) = 0\)
  • \(\lim_{x \to +\infty} F(x) = 1\)

There is technically one more point needed as well, but we won't test you on it:
  • \(F(x)\) is right-continuous, meaning that \(\lim_{y \to x^+} f(y) = f(x)\) for all \(x \in \mathbb{R}\). In other words, when I approach any point \(x\) from the right, the limit is \(f(x)\). 
What is a random variable
A random variable is a function from the outcomes of a probability space to the real numbers.

(So, it is neither random, nor a variable, despite its name! It's a deterministic function.)
