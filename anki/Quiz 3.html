Conditional Expectation
Given a discrete random variable \(X\) and an event \(E\) with \(\mathbb{P}[E] > 0\), we define
\[\mathbb{E}[X \mid E] = \sum_{x \in R_x} x \cdot \mathbb{P}[X=x \mid E]\]
where \(R_X\) is the range of \(X\).

(We will not test you on the definition for continuous random variables, but if \(E\) corresponds to a set of values \(X\) can take on with \(\mathbb{P}[E] > 0\), we can define \(f_{X \mid E}(x) = \frac{f(x)}{\mathbb{P}[E]}\) if \(x \in E\), and 0 otherwise. Then,
\[\mathbb{E}[X \mid E] = \int_{-\infty}^\infty x \cdot f_{X \mid E}(x) dx\]
However, to define this more generally we need a joint distribution.)
Expectation of Product of Discrete Random Variables (dependent and independent)
Given any two discrete random variables \(X,Y\):
\[\mathbb{E}[X \cdot Y] = \sum_{x \in R_x} \sum_{y \in R_y} x \cdot y \cdot \mathbb{P}[X=x,Y=y]\]If \(X,Y\) are independent, then in both the discrete and continuous case,
\[\mathbb{E}[X \cdot Y] = \mathbb{E}[X] \cdot \mathbb{E}[Y]\]We will not test you on this, but in case you are curious, if \(X,Y\) are continuous, then
\[\mathbb{E}[X \cdot Y] = \int_{-\infty}^\infty \int_{-\infty}^\infty x \cdot y \cdot f_{X,Y}(x,y) dx fy\]
where \(f_{X,Y}\) is the joint density function of \(X\) and \(Y\). This is needed in the case that \(X,Y\) are not independent.
Expected Value of a Continuous Random Variable
For a continuous random variable \(X\) with PDF \(f_X\), we define
\[\mathbb{E}[X] = \int_{-\infty}^\infty x \cdot f_X(x) dx\] 
Expected Value of a Discrete Random Variable
For a discrete random variable \(X\),
\[\mathbb{E}[X] = \sum_{\omega \in \Omega} X(\omega) \cdot \mathbb{P}(\omega)\]We also showed that \(\mathbb{E}[X] = \sum_{x \in R_X} x \cdot \mathbb{P}[X=x]\), where \(R_X\) is the range of \(X\). This is often more useful. 
Harmonic Number and Asymptotics
The \(n\)th harmonic number, \(H_n\), is defined \(H_n = 1+\frac{1}{2}+\frac{1}{3} + \dots = \sum_{i=1}^n \frac{1}{i}\).

We can approximate \(H_n \approx \ln(n)\). More precisely, for any \(n \ge 2\), we have \(\ln(n) < H_n < \ln(n) + 1\). 
Independence for discrete random variables
Two discrete random variables \(X,Y\) are independent if for all \(x,y \in \mathbb{R}\), the events \([X=x],[Y=y]\) are independent.

Similarly, a collection of random variables \(X_1,\dots,X_n\) are mutually independent if for all \(x_1,\dots,x_n \in \mathbb{R}\), the events \([X_1=x_1],[X_2=x_2],\dots,[X_n=x_n]\) are mutually independent. 

An analogous definition holds for \(k\)-wise independence.
Independence of random variables
Two random variables \(X,Y\) are independent if for all \(x,y \in \mathbb{R}\), the events \([X \le x]\) and \([Y \le y]\) are independent

Similarly, random variables \(X_1,\dots,X_n\) are mutually independent if for all \(x_1,x_2,\dots,x_n \in \mathbb{R}\), the events \([X_1 \le x_1],[X_2 \le x_2], \dots, [X_n \le x_n]\) are mutually independent.

An analogous definition holds for \(k\)-wise independence.
Independent events
Two events \(A,B\) are independent if \(\mathbb{P}[A \cap B] = \mathbb{P}[A] \cdot \mathbb{P}[B]\).
Indicator Random Variable
Given an event \(E\), the indicator random variable \(\mathbb{I}[E]\) is 1 when \(E\) occurs and 0 otherwise. So, \(\mathbb{I}[E]\) is a Bernoulli with success probability \(\mathbb{P}[E]\).

Often, given a collection of \(n\) events, we will create random variables \(X_1,\dots,X_n\) so that a random variable \(X\) is equal to \(X=\sum_{i=1}^n X_i\).
LOTUS (Law of the Unconscious Statistician)
Given any discrete random variable \(X\) and any function \(g: \mathbb{R} \to \mathbb{R}\),
\[\mathbb{E}[g(X)] = \sum_{x \in R_x} g(x) \cdot \mathbb{P}[X=x]\]
where \(R_X\) is the range of \(X\). For a continuous random variable \(X\) with PDF \(f_X\), we similarly have
\[\mathbb{E}[g(X)] = \int_{-\infty}^\infty g(x) f_X(x) dx\]
Law of Total Expectation or Tower Rule
The most general form of the Law of Total Expectation is as follows: for any two random variables \(X,Y\) (which may be continuous or discrete), we have \(\mathbb{E}[X] = \mathbb{E}[\mathbb{E}[X \mid Y]]\). This is sometimes called the Tower Rule.

Remember that \(\mathbb{E}[X \mid Y]\) is itself a random variable that given \(\omega \in \Omega\) with \(Y(\omega)=y\) takes value \(\mathbb{E}[X \mid Y=y]\). 

For discrete random variables and in many cases, it is often more convenient to use the following formulation. Given a partition \({\cal P} = P_1,\dots,P_n\) of \(\Omega\) and a random variable \(X\), \(\mathbb{E}[X] = \sum_{i=1}^n \mathbb{E}[X \mid P_i]\mathbb{P}[P_i]\). \({\cal P}=\{P_i\}_{i \in \mathbb{N}}\) can also be a countable partition of the sample space so that \(\mathbb{E}[X] = \sum_{i \in \mathbb{N}} \mathbb{E}[X \mid P_i] \mathbb{P}[P_i]\). 
Law of Total Variance
Given two random variables \(X,Y\), we have:
\(\text{Var}(X) = \mathbb{E}[\text{Var}(X \mid Y)] + \text{Var}(\mathbb{E}[X \mid Y])\)

Let's parse this equation. Both \(\text{Var}(X \mid Y)\) and \(\mathbb{E}[X \mid Y]\) are random variables, so we can take their expectation and variance. Given a sample \(\omega\) with \(Y(\omega)=y\), \(\text{Var}(X \mid Y)\) takes the value \(\text{Var}(X \mid Y=y)\) and \(\mathbb{E}[X \mid Y]\) takes the value \(\mathbb{E}[X \mid Y=y]\).

The first term is the part of the variance which is not explained by \(Y\): it is the uncertainty that remains after knowing \(Y\). The second part is the part of the variance explained by \(Y\). 

To gain some intuition about what this means, consider two extreme cases.
1. Suppose \(X=Y\). Then, given the value of \(Y\), \(X\) is fixed, so it has 0 variance. Therefore, in this case, \(\mathbb{E}[\text{Var}(X \mid Y)] = 0\): there is no uncertainty remaining once we know \(Y\). But, since \(\mathbb{E}[X \mid Y=y] = y\), this random variable is equivalent to \(Y\), which is equivalent to \(X\), so it is simply \(\text{Var}(X)\) (all of the variance is explained by \(Y\)) and the equation holds. 
2. Suppose \(X,Y\) are independent. Then, given the value of \(Y\), we learn nothing about \(X\), so \(\mathbb{E}[\text{Var}(X \mid Y)] = \text{Var}(X)\). And the expected value of \(X\) does not change depending on the value of \(Y\), so \(\mathbb{E}[X \mid Y]\) is a random variable always taking a single value and thus it has 0 variance, and again the equation holds. 

This is sometimes known as Eve's law. 
Linearity of Conditional Expectation
Given any random variables \(X_1,\dots,X_n\), any constants \(c_1,\dots,c_n \in \mathbb{R}\), any \(c \in \mathbb{R}\), and any event \(A\),
\[\mathbb{E}[c+\sum_{i=1}^n c_iX_i \mid A] = c+\sum_{i=1}^n c_i\mathbb{E}[X_i \mid A]\]Remember, this is because conditional distributions are distributions. So all rules we have learned hold for conditional distributions. 
Linearity of Expectation
Given any \(k\) random variables \(X_1,\dots,X_k\) and any \(c_1,\dots,c_{n} \in \mathbb{R}\) and \(c \in \mathbb{R}\), we have
\[\mathbb{E}[c+\sum_{i=1}^n c_iX_i] = c+ \sum_{i=1}^n c_i\mathbb{E}[X_i]\]
Mutual independence
A set of events \(A_1,\dots,A_n\) is mutually independent if for every \(S \subseteq [n]\),
\[\mathbb{P}[\bigcap_{i \in S} A_i] = \prod_{i \in S}\mathbb{P}[A_i]\]
Pairwise independence for a collection of events
A collection of events \(A_1,\dots,A_n\) is pairwise independent if for all distinct \(i,j \in [n]\), \(A_i\) and \(A_j\) are independent.
Properties of Variance
For \(a,b \in \mathbb{R}\) and a random variable \(X\), we have \(\text{Var}(aX+b) = a^2\text{Var}(X)\).

For \(X_1,\dots,X_n\) pairwise independent, we have \(\text{Var}(\sum_{i=1}^n X_i) = \sum_{i=1}^n \text{Var}(X_i)\). 
Standard Deviation
Given a random variable \(X\), the standard deviation \(\sigma\) is defined \(\sigma = \sqrt{\text{Var}(X)} = \sqrt{\mathbb{E}[(X-\mathbb{E}[X])^2]}\).
Variance
The variance of a random variable \(X\) is \(\mathbb{E}[(X-\mathbb{E}[X])^2]\). We showed this is equal to the often more convenient expression \(\mathbb{E}[X^2] - \mathbb{E}[X]^2\). 
\(k\)-wise independence
A set of events \(A_1,\dots,A_n\) are \(k\)-wise independent if every subset of at most \(k\) of these events are mutually independent.
